# 📝 Explicación: Formulario tecnologico 💜

Mi versión utiliza una paleta de colores más tecnológica. Estos son los ajustes técnicos que hice:

### 1. Identidad Visual Renovada
* **Color Maestro:** Remplace el color principal por un **violeta brillante** (`#8b5cf6`). Este color se aplica al encabezado y a los elementos de interacción.
* **Curvaturas:** Mantuve el radio de **20px** para que el diseño no se vea "cuadrado" o antiguo, dándole un toque de aplicación móvil moderna.

### 2. Implementación del Campo Edad
* Inserte el código HTML necesario para capturar la edad del participante.
* Es un campo de tipo numérico que ayuda a filtrar a los usuarios, asegurando que solo personas entre **15 y 99 años** puedan participar. Es una forma sencilla de validar datos sin usar JavaScript todavía.


### 3. Efecto de Sombra Brutal
* Use un color negro con transparencia (`rgba(0,0,0,0.7)`) para que la sombra se vea natural pero "pesada". 
* El resultado es un botón que parece tener un impacto visual masivo cada vez que el usuario intenta hacer clic en él.
